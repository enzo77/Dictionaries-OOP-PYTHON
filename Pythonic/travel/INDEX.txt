🎯 CONCEPTOS CUBIERTOS
════════════════════════════════════════════════════════════════════════════

Context Managers:
  ✓ __enter__() y __exit__()
  ✓ Gestión automática de recursos
  ✓ COMMIT/ROLLBACK automático
  ✓ Transacciones ACID

Mixins:
  ✓ Herencia múltiple
  ✓ Composición de comportamiento
  ✓ MRO (Method Resolution Order)
  ✓ Ventajas sobre herencia profunda

SQLite:
  ✓ Conexiones seguras
  ✓ Parámetros en queries
  ✓ Protección SQL Injection
  ✓ Creación de tablas

Patrones de Diseño:
  ✓ Repository Pattern
  ✓ CRUD operations
  ✓ Separación de responsabilidades
  ✓ Herencia vs Composición
